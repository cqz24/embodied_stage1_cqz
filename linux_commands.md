# Linux 常用命令记录

## 文件和目录操作
- pwd：显示当前路径
- ls：列出目录内容
- cd：切换目录
- mkdir：创建目录
- touch：创建空文件
- cp：复制文件或目录
- mv：移动或重命名
- rm：删除文件或目录（小心使用）

## 文件内容查看
- cat：显示文件全部内容
- less：分页查看
- head / tail：查看文件开头/结尾
- grep：搜索文本

## 权限管理
- chmod：修改权限
- chown：修改所有者

## 打包压缩
- tar -czvf 包名.tar.gz 文件夹/：打包并压缩
- tar -xzvf 包名.tar.gz：解包

## 其他
- echo：打印内容或写入文件
- > 和 >>：重定向输出
- |：管道，将前一个命令的输出作为后一个命令的输入
